# 📝 Digital Literacy Portfolio - CSE0001

## 📌 Project Overview
This repository contains my **Digital Literacy Portfolio** for the CSE0001 course. Developed under the **Student Digital Ambassador** scenario, it includes five distinct tasks—one for each course module. These tasks demonstrate proficiency in digital tools, maintaining a professional online presence, utilizing coding platforms, practicing email etiquette, and understanding cybercrime awareness.
